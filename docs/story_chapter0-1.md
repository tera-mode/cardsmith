「マテリアル」→「写石」に変更

### 領域探索モード仕様

```
各領域：5ステージ
各ステージで取得：2種類のカード × 2枚ずつ = 計4枚
全コンプリート：6領域 × 5ステージ × 4枚 = 120枚
```

| ステージ | 取得カードのコスト帯 |
|---------|------------------|
| ステージ1 | 各領域の最低コスト2種類（C級） |
| ステージ2 | 各領域の低コスト2種類（C-R級） |
| ステージ3 | 各領域の中コスト2種類（R級） |
| ステージ4 | 各領域の中高コスト2種類（R-SR級） |
| ステージ5 | 各領域の高コスト2種類（SR級） |

### 既存実装からの変更点

| 既存 | 変更後 |
|------|------|
| チュートリアルクエスト | **廃止**。第0章のチュートリアルに置き換え |
| 1ステージで1種類のカード取得 | **1ステージで2種類×2枚 = 4枚取得** |
| 既存ストーリーモード | **第0-1章で書き直し** |

## 用語ガイド

### 重要用語

| 用語 | 読み | 意味 |
|------|------|------|
| **写石** | しゃせき | 人のスキルを写した石。カードの素材 |
| **空の写石** | からのしゃせき | 転写前の石。写畜が自分のスキルを写す対象 |
| **写畜** | しゃちく | スキルを写石に転写して売る庶民。やや蔑称的 |
| **写石商** | しゃせきしょう | 写畜の公式呼称（上品な言い方） |
| **コンセプト・スミス** | - | 写石なしでカードを作れる伝説の存在 |
| **大鍛冶閥** | おおかじばつ | 上層カードスミスの財閥 |

### 「写す」と「削る」の使い分け

| 動詞 | 使う文脈 |
|------|------|
| **写す** | 技術的・客観的「スキルを空の写石に写す」「写しすぎた」 |
| **削る** | 感情的・主観的「命が削られる」「削られてきた人生」 |

両方とも世界観として正当。混在使用OK。

## 掛け合いシーンの演出ルール

### 基本ルール

```
[キャラ・位置イン / 表情ID]
セリフ
[キャラ・退場]
[次のキャラ・位置イン / 表情ID]
セリフ
...
```

- **左イン**：左からフェードイン
- **右イン**：右からフェードイン
- **中央イン**：中央フェードイン（主人公単独セリフなど）
- **退場**：フェードアウト
- **/ 表情ID**：使用する表情（calm, smile, serious等）

### 1人だけが話すとき
キャラを画面に出したまま、表情だけ切り替える。

```
[ブリギッタ・左イン / calm]
ブリギッタ「ボウズ」
ブリギッタ「（serious表情に切替）よく聞け」
ブリギッタ「（soft表情に切替）……いや、何でもない」
```

### 思考・心情
キャラ立ち絵の前、または立ち絵を表示せず、思考枠で表示。

```
[主人公（思考）]
「神様が……？」
```
