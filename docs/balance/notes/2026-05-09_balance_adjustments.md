# 2026-05-09 バランス調整：先行後攻補正・横攻撃・強打スキル

## Plan（仮説）

自動対戦シミュレーター（750試合, seed=42）の結果を受け、以下3点の調整を実施。

### 1. 先行後攻補正 (L4)
- **問題**: 同デッキミラーで先攻勝率 ~74%（目標 48〜52%）
- **仮説**: 後攻に初期手札+1枚を与えることでテンポ差を補正できる
- **調整案**: 後攻プレイヤーの初期手札を `INITIAL_HAND_SIZE + 1 = 4枚` に変更

### 2. 低コストユニットの横攻撃解禁 (L1)
- **問題**: cost 4-6の近接ユニットが前1マスしか攻撃できず、ラインを突破されると何もできない
- **仮説**: `FWD_SIDES`（前+左右）に変更することで盤面活用の幅が広がる
- **対象カード**: sei_noa, mei_cal, shin_hina, en_koko, en_ron, sou_kaito, sou_miu, kou_mk01, sei_liese, sou_nadja

### 3. 強打スキル追加 (L3)
- **問題**: 先にベース攻撃されると巻き返しが難しい
- **仮説**: 各系統の低コストカードに「前線から3ダメージ」スキルを付与で逆転要素を追加
- **スキル定義**: `strong_blow` (id: 'strong_blow') - activated, 最前線から敵陣地に3ダメージ (2回)
- **付与カード**: sei_noa, mei_cal, shin_hina, en_ron, sou_nadja, kou_mk01

## Do（実装）

- `src/lib/game/skills/strong_blow.ts` 新規作成
- `src/lib/game/cards.ts`:
  - `FWD_SIDES` 定数追加
  - cost 4-8 近接ユニット 10枚のattackRangeを `FWD_SIDES` に変更
  - sei_noa, mei_cal, shin_hina, en_ron, sou_nadja, kou_mk01 に `strong_blow` スキル付与
- `src/contexts/GameContext.tsx`: AI初期手札を `INITIAL_HAND_SIZE + 1` に変更
- `src/lib/game/sim/runMatch.ts`: 後攻プレイヤーの初期手札+1 適用

## Act（次の方針）

### 成功点
- 先行後攻補正は機能し、同デッキミラーの先攻有利が改善（~74% → ~64%）
- sei_5の突出度が緩和（78% → 70.7%）

### 課題
- **sei_1の大幅低下（59.3% → 43.3%）** が想定外
  - 原因仮説1: sei_noa への `strong_blow` スキル付与でAI評価の `skillPotential` スコアが変化し、行動選択が変わった
  - 原因仮説2: 後攻+1手札で対戦相手が有利になった（sei_1は低コスト構成で後攻ボーナスの恩恵を受けにくい）
- **同デッキミラーがまだ 53-70%（目標 48-52%）** で先攻有利が残る

### 次サイクルの方針
- sei_1 の個別調査（strong_blow スキルなしの場合との比較）
- 後攻補正の数値検討（+1 → +2 手札、または代わりに後攻 baseHp+1）
- 全6系統の Element Balance テストを実施して横断比較
