## 表記凡例

| 略号 | 意味 |
|---|---|
| 前1 | 前方1マスに隣接攻撃 |
| 前+左右 | 前方・左・右の3方向に隣接攻撃 |
| 4方向 | 前後左右の4方向に隣接攻撃 |
| 前nJ-射 | 前方n射程（弓兵型・貫通なし） |
| 周8 | 周囲8方向（AOE） |
| なし | 攻撃手段なし |
| 前1-2J移 | 前1マスまたは前2マスジャンプ（どちらか選択可） |
| (n) | スキル使用回数（n=数字 or ∞） |
| [オーラ] | 常時発動するパッシブオーラ（uses非適用） |

### ルール変更
- **前2Jユニット**（セラフィム・ノクト・ルキフェル・ティターニア・ファイ・リン・アスカ）は「前1または前2マスジャンプ」が選択可能になった
- **強打**のダメージ: 3→2。新スキル**重撃**（3ダメ）を焔アスカに追加
